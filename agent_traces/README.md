# Agent Traces

This folder is for the 10 Part 2 agent trace JSON files generated by:

```bash
cd script
python agent_controller.py --model qwen2.5:7b --top-k 5
```

Expected files:

```text
all_traces.json
trace_summary.md
```

`all_traces.json` contains the 10 task traces. Each trace records:

- decision points
- selected tools
- reasons for tool choices
- retrieval results
- intermediate tool outputs
- final answer
- latency

Use `--plan-only` only to preview trace structure without model calls. Final submitted traces should be generated with `qwen2.5:7b`.
